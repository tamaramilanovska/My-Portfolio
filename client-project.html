CLIENT PROJECT

This was a team project where we had an assignment to create a web page for Blood Donors. Our client wanted a simple page with all the needed informations. My assignment was to create the Contact and About Us Pages. We had a lot of fun making this project, our team had a great communication and everyone was hardworking and dedicated.

We had three phases

  • First Phase: We spend most of the time doing research, gathering informations, trying some designs and planning. After we had everything that was needed we started implementing it in HTML and CSS.
  • Second Phase: In the second phase we started writing Javascript.
  • Third Phase: In the third phase we started implementing our project in Wordpress.
